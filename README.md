# PixelCraft

A modern and responsive Digital Agency website built using React.js and Tailwind CSS. This project showcases a clean user interface, responsive layouts, and reusable React components.

## 🚀 Live Demo

https://pixelcraft-delta.vercel.app

## 📂 GitHub Repository

https://github.com/keerthana-25-2004/Pixelcraft

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Vite
- React Icons
- JavaScript (ES6+)
- HTML5
- CSS3

---

## ✨ Features

- Responsive Navigation Bar
- Mobile Menu using React useState
- Hero Section
- Services Section
- Portfolio Section
- Contact Form with Validation
- Footer
- Smooth Navigation
- Responsive Design for Mobile, Tablet and Desktop

---

## ⚙️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/keerthana-25-2004/Pixelcraft.git
```

2. Navigate to the project folder

```bash
cd Pixelcraft
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

5. Open the application in your browser

```
http://localhost:5173
```
